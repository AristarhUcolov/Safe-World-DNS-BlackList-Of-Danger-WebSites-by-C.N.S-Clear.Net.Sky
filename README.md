### How to Use

#### Option 1: Pi-hole
1. Open Pi-hole admin panel
2. Navigate to **Group Management** ‚Üí **Adlists**
3. Add the RAW link above
4. Update gravity: `pihole -g`

#### Option 2: AdGuard Home
1. Open AdGuard Home dashboard
2. Go to **Filters** ‚Üí **DNS blocklists**
3. Click **Add blocklist**
4. Paste the RAW link and save

#### Option 3: NextDNS
1. Log in to NextDNS
2. Go to **Security** ‚Üí **Blocklists**
3. Add custom blocklist with the RAW link

#### Option 4: Manual DNS Server
Use the blocklist with BIND, Unbound, or other DNS servers by importing the list and configuring appropriate block responses.

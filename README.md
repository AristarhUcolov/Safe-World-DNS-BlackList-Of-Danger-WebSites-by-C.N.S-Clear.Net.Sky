# Safe World DNS Blacklist

This repository is designed to curate a blacklist of dangerous websites to protect the users from harmful content.

## Introduction

The DNS Blacklist is a mechanism to prevent access to malicious websites. This list is maintained by users and security experts.

## How to Use

To implement the blacklist, you will need to configure your DNS settings to point to a DNS server that respects this blocklist.

## Contribution

To contribute, fork the repository and add your own entries, then create a pull request for review.
